# FUTURE_ML_02
****Support Ticket Classification Prioritization**
This project builds a machine learning system to automatically classify support tickets and assign priority levels.

**Features**

Text cleaning and preprocessing
TF-IDF feature extraction
Ticket classification using Linear SVM
Priority prediction (High / Medium / Low)
Model evaluation (Accuracy, Precision, Recall)
Confusion Matrix visualization

**Results**

Achieved ~85% accuracy
Strong performance across multiple ticket categories

**Business Use**

Faster ticket categorization
Improved response time
Reduced manual effort for support teams

**Tools Used**

Python
Pandas, NumPy
Scikit-learn
Matplotlib

**Conclusion**

This system helps automate support operations and improves efficiency in handling customer queries.
